# Interactive Map Project

This project provides an interactive map that allows users to mark two positions and calculate the geodesic distance between them. It utilizes the Leaflet library for mapping and a geospatial library for distance calculations.

## Project Structure

```
interactive-map-project
├── src
│   ├── index.html       # HTML structure for the interactive map
│   ├── main.js          # JavaScript code for map interactions and distance calculations
│   └── styles.css       # CSS styles for layout and appearance
├── package.json         # npm configuration file with project dependencies
└── README.md            # Documentation for the project
```

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd interactive-map-project
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Open the project:**
   Open `src/index.html` in your web browser to view the interactive map.

## Usage Guidelines

- Click on the map to place two markers. 
- The distance between the two markers will be calculated and displayed below the map.
- Ensure that you have an active internet connection to load the Leaflet library.
